# AI Journey — 6-Month Path to AI Engineer

A structured, project-driven self-study program covering Python engineering,
backend development, machine learning, LLM systems, and production deployment.

## Roadmap

| Month | Focus | Project |
|---|---|---|
| Month 1 | Python + Git + FastAPI basics | File Organizer |
| Month 2 | FastAPI deep + PostgreSQL | Task Manager API |
| Month 3 | ML fundamentals + scikit-learn | Ticket Classifier API |
| Month 4 | LLM & GenAI + RAG systems | Domain RAG Chatbot |
| Month 5 | Docker + Cloud deployment | Production AI Assistant |

## Tech stack being built
Python · FastAPI · PostgreSQL · scikit-learn · PyTorch · 
LangChain · ChromaDB · Docker · AWS

## Approach
Every month builds on the previous one. Projects are not standalone 
tutorials — they chain into a single production AI application by Month 5.
